介绍

FastBuilder Cloud 是运行在 BuShe BDocker (由 Docker 驱动) 的服务版 FastBuilder，通俗来说，在启用此服务后，您只需要在 Minecraft 中使用命令 /connect [我们提供给您的地址] 即可连接到 FastBuilder Pro，无需任何替他配置。

目前此服务正在公测。
